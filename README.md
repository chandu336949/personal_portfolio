# 💻 Personal Portfolio Website

A sleek, responsive, single-page developer portfolio website designed to showcase my projects, core technical skills, and problem-solving journey.

[![Live Demo](https://img.shields.io/badge/Demo-Live%20Preview-brightgreen?style=for-the-badge&logo=vercel)](https://chandusrikuramdasu-portfolio.vercel.app/)

---

## 📖 About The Project

This website serves as my digital footprint and interactive portfolio. Built using prompt-driven development via **Lovable AI**, the site outlines my academic background, technical competencies, and standout projects in an intuitive, recruiter-friendly design. 

The primary goal of this repository is to demonstrate how I combine traditional frontend development frameworks with next-generation AI workflows to build high-performance user interfaces.

---

## ✨ Features

*   **⚡ Modern Hero Section:** A striking introduction showcasing my core development identity and instant Call-to-Actions (CTAs).
*   **📂 Interactive Project Showcase:** A clean grid featuring detailed summaries, live links, and repository access to my major applications.
*   **🛠️ Dynamic Skill Grid:** Categorized overview of core technologies, tools, and conceptual fundamentals.
*   **📄 Direct Resume Access:** Streamlined integration allowing recruiters to download my professional resume with a single click.
*   **📱 Fluid Responsiveness:** Pixel-perfect adjustments engineered using Tailwind CSS to look stunning across mobile, tablet, and desktop environments.
*   **✉️ Seamless Contact Gateway:** A clean, accessible layout designed for immediate feedback and messaging.

---

## 🛠️ Tech Stack

*   **Core:** React, TypeScript, Vite
*   **Styling:** Tailwind CSS, HTML5, CSS3
*   **Build & Configuration Platform:** Lovable AI
*   **Hosting:** Lovable Hosting
